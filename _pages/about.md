I am a researcher interested in how computational methods can help us ask better scientific questions, build more reliable workflows, and turn complex data into useful evidence.

My work sits around three themes:

- **AI for science**: using machine learning, language models, and data systems to support discovery.
- **Reproducible research**: building workflows that make analysis easier to inspect, rerun, and extend.
- **Scientific communication**: turning technical results into clear papers, figures, talks, and tools.

This website collects my publications, selected projects, research notes, and CV. The current version is a clean starting point; the next step is to add my real affiliation, profile photo, publication BibTeX, and project details.
